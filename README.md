This repository contains the complete source code for the Project Bayanihan website, a nonprofit initiative focused on supporting schools and underserved communities in Sudipen, Philippines. The site is fully coded from scratch using HTML, CSS, and JavaScript, with zero reliance on site builders. Features include:

Mission and vision sections

Impact counters (schools, students, volunteers, supplies)

Project highlights

Volunteer sign-up form (mailto + clipboard)

Fully responsive design
